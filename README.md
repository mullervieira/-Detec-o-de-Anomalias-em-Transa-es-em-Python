#  Desafio de Machine Learning

## Sobre o projeto

Este projeto foi desenvolvido como parte de um desafio de Machine Learning.

## Objetivo

Construir um modelo de classificação utilizando técnicas de ciência de dados.

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SHAP
- Google Colab

## Etapas realizadas

- Análise exploratória dos dados
- Tratamento de valores ausentes
- Balanceamento das classes
- Treinamento do modelo XGBoost
- Ajuste de hiperparâmetros com GridSearchCV
- Avaliação do modelo
- Explicabilidade utilizando SHAP

## Arquivos

- `desafio.ipynb` — Notebook contendo todo o desenvolvimento do projeto.
